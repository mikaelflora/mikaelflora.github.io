Contexte du projet

Dans le cadre de mon stage au laboratoire MIO on m'a chargé de mettre en place une solution de récupération de donné météorologique. Jusqu'à présent les stations météo qu'ils utilisent stoque les données localement et il faut qu'une personne aille les récupérer sur place.

Détails

La solution sur laquelle j'ai travaillé se charge de récupérer les données des stations météos, de les mettre en forme puis de les retransmettre sur un serveur distant. J'ai développé un programme Python qui s'installe sur une carte programmable de type Raspberry ou Beagle Board (OS de base Debian). Une fois les données récupérées, un tâche s'exécute régulièrement pour les envoyer sur un serveur FTP.
