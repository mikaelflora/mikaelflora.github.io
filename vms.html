Contexte du projet

Dans le cadre de mon travail chez Axe-e-Santé j'ai été amené a créer des VMs pour les clients. Cette tâche de création peut prendre du temps et nécessite de la rigueur pour éviter les erreurs (création, installation et paramétrage). J'ai donc travaillé sur une solution pour automatiser ces tâches répétitives

Détails

J'ai choisi d'utiliser Vagrant/VirtualBox pour créer les VM puis Ansible pour gérer leurs configurations. Une fois les tâches terminées il ne reste plus qu'à convertir les VM aux formats appropriés pour le client (ova pour Hyper-V / vmdk pour VMWare).

Ansible a aussi pu être utilisé au travers d'une API pour gérer la configuration de postes physiques ou encore le déploiement de mise à jours sur les infrastructures existantes.
