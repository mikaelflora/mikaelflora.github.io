Contexte du projet

Lors de ma formation chez OpenClassrooms j'ai participé à l'administration d'un serveur Discord. Je l'ai administré depuis sa création jusqu'à qu'il avoisine les 400 membres. Pour accueillir les nouveaux membres j'ai mis en place un Bot.

Détails

Implémentation d'un bot Discord en Python via la lib discord.py et l'utilisation d'une base de données SQLite.

  • Fonctionnalités
  • Core : outils de maintenance du bot (gestion du bot via ses plugins, latence, logs)
  • Admin : outils d'administration (gestion du serveur, gestion des membres ...)
  • Poll : système de sondage simple
  • Help : affichage d'une aide sur la syntaxe "Markdown Discord"
  • Welcome : gestion du message de bienvenue en fonction de variables définies par l'administrateur du serveur

Pour accéder au code source du bot : Dépôt Bitbucket
